How to use

Quick start

  1. Name your card layers using the table below.
  2. Select the card frames you want to populate.
  3. Hit Populate Products for instant results from the built-in catalog.
Tip: Select a parent frame to update all product cards inside it at once.
Catalog: 210 products, 6 categories (HP, Lenovo, 3M, Bosch, DeWalt, Littmann, etc.).

Layer names

Rename layers in your card to match these exactly (case-sensitive). Only the two marked required are needed; add any others you want.
